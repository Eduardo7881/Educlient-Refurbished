Requirements to run Educlient

The Educlient was projected to run optimized on Minecraft 1.8

Minimum Requirements:

  • Processor: Intel Core 2 Duo or equivalent
  • RAM Memory: 2GB
  • Video card: Intel HD Graphics 2000 or higher (with OpenGL 2.0)
  • Storage: 100MB

Recommended Requirements:

  • Processor: Intel Core i5 or AMD Ryzen 3
  • RAM Memory: 4GB+
  • Video card: NVIDIA GeForce GTX 750 or higher
  • Storage: 2GB+
